# palo.sk

Bilingual personal website of Dominik Paľo — software engineer, maker, and
lecturer at MakerSpace Bratislava.

## Development

```bash
npm install
npm run dev
```

## Production

```bash
npm run build
```

The static output is written to `dist/`. Pushes to `main` are deployed to
[palo.sk](https://palo.sk) through GitHub Pages.
